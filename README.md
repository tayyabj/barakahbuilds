# Barakah Builds

Updated static website for a 3D printing hobby brand in Irving, TX.

## Pages
- About (`index.html`)
- Gallery (`gallery.html`)
- Order (`order.html`)
- Contact (`contact.html`)

## Included
- Black + gold brand styling
- More colorful accents for product energy
- Responsive navigation
- Your provided logo (`logo.png`)

## Run locally
```bash
python3 -m http.server 8000
```

Open:
http://localhost:8000

## Deploy to GitHub Pages
Upload the full folder contents to the root of your repo, then enable:
Settings → Pages → Deploy from a branch → `main` → `/ (root)`
